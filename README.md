# Svelte Electron template

## Usage

```bash
npx degit "maxatwork/svelte-electron-template" svelte-electron
cd svelte-electron
npm install
```

### Storybook

```
npm run storybook
```

Opens browser with Storybook.

### Electron app with live reload

```
npm run start:dev
```

### Packaging application

```
npm run package
```

Packaged app will be in `./dist/` folder.
